Principais formatações

Negrito / Destaque

Pra utilizarmos negrito, podemos utilizar <b> ficando dessa forma:
Negritooo. Porém é uma forma não semântica.

Nesta frase, usando a tag <strong>, daremos destaque a um termo. De forma semântica.

Itálico / Ênfase

Pra utilizarmos itálico, podemos utilizar <i> ficando dessa forma:
Itália! Porém também é uma forma não semântica.

Nesta frase, temos um termo em ênfase usando a tag EM (semântica)

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e Pequeno

Estamos criando um texto grande e um texto pequeno.

Texto deletado

Podemos marcar que um texto como excluído para dar uma ênfase que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar ênfase e indicar que ele foi adicionado depois.

Também existe o sublinhado com a tag U.(não semântica)
